## Repository Structure

### Input
Basic programs related to user input handling and introductory programming concepts.

### Loops
Programs based on:
- for loops
- while loops
- nested loops
- loop-based problem solving

### Pattern Printing
Pattern-based problems used to improve logical thinking and loop understanding.

Examples:
- Star Patterns
- Number Patterns
- Pyramid Patterns
- Hollow Patterns

### Module2Conditions
Programs based on:
- if-else statements
- nested conditions
- switch cases
- decision making

### Functions
Programs demonstrating:
- Function declaration
- Function definition
- Function calling
- Parameter passing
- Return values

### Sec_10_Pointers
Practice programs covering:
- Pointer basics
- Pointer arithmetic
- Call by reference
- Memory concepts

### MISC
Additional problems and concepts that do not fit into a specific category.
